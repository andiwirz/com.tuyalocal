ⓘ Wo finde ich Device ID & Local Key?
1. Vorbereitung: App-Konto verknüpfen
  1. Loggen Sie sich in die Tuya Developer Plattform ein und öffnen Sie Ihr Projekt unter Cloud > Development.
  2. Wechseln Sie zum Reiter Devices und wählen Sie Link Tuya App Account.
  3. Klicken Sie auf Add App Account und scannen Sie den QR-Code mit Ihrer Smart Life / Tuya App (unter „Profil" > QR-Code-Scanner oben rechts).
  4. Bestätigen Sie die Verknüpfung in der App. Ihre Geräte erscheinen nun unter dem Tab All Devices.
2. Device ID finden
  1. Gehen Sie im Projekt zu Cloud > Devices > All Devices.
  2. Suchen Sie Ihr Gerät in der Liste. Die Device ID steht direkt unter dem Gerätenamen. Kopieren Sie diese.
3. Local Key über den API Explorer auslesen
  1. Klicken Sie in der linken Seitenleiste auf Cloud > API Explorer.
  2. Wählen Sie im linken Menü: Smart Home Management System > Device Management > Query Device Details.
  3. Wählen Sie oben rechts das korrekte Rechenzentrum (z. B. „Central Europe Data Center").
  4. Fügen Sie Ihre Device ID in das Eingabefeld ein und klicken Sie auf Submit Request.
  5. Suchen Sie im JSON-Antwort-Fenster nach dem Feld local_key. Dieser Wert ist Ihr lokaler Schlüssel.
Trotzdem fortfahren (ohne Verbindungstest)
